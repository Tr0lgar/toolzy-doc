Usage examples

Discover how to integrate Toolzy into your projects with these practical examples.

Basic example

import { randomInt } from 'toolzy';

// Generate a random number for a dice game
const diceRoll = randomInt(1, 6);
console.log(`You rolled: ${diceRoll}`);

// Generate a simple random ID
const randomId = randomInt(1000, 9999);
console.log(`Generated ID: ${randomId}`);

Using in TypeScript

import { randomInt } from 'toolzy';

interface GameSettings {
minPlayers: number;
maxPlayers: number;
}

const gameConfig: GameSettings = {
minPlayers: 2,
maxPlayers: 8
};

const playerCount: number = randomInt(
gameConfig.minPlayers,
gameConfig.maxPlayers
);

console.log(`Number of players: ${playerCount}`);
