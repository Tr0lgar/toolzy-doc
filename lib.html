Library Documentation

Discover all functions available in Toolzy.

Random Number Generation

randomInt(min, max) → number
▼
Generates a random integer between the minimum and maximum values (inclusive).

Parameters:

  • min (number) - The minimum value
  • max (number) - The maximum value

Returns:

A random integer between min and max (inclusive)

Example:
import { randomInt } from 'toolzy';

const dice = randomInt(1, 6);
console.log(dice); // 1, 2, 3, 4, 5, or 6

String Manipulation

capitalizeFirst(str) → string
▼
Capitalizes the first letter of the given string.

Parameters:

  • str (string) - The string to be processed

Returns:

The input string with its first character capitalized

Example:
import { capitalizeFirst } from 'toolzy';

const result = capitalizeFirst('hello');
console.log(result); // "Hello"
slugify(text) → string
▼
Converts a given string into a URL-friendly slug by normalizing characters, removing diacritics, and replacing spaces and special characters with a hyphen.

Parameters:

  • text (string) - The input string to be slugified

Returns:

The slugified version of the input string

Example:
import { slugify } from 'toolzy';

const title = 'Café de la Paix!';
console.log(slugify(title)); // "cafe-de-la-paix"
truncate(str, maxLength, suffix?) → string
▼
Truncates a string to a specified maximum length, optionally adding a suffix if truncation occurs.

Parameters:

  • str (string) - The string to be truncated
  • maxLength (number) - The maximum allowed length of the final string, including suffix
  • suffix? (string) - A string to append if truncation occurs (default is ...)

Returns:

The truncated string, with suffix if truncation occurred

Example:
import { truncate } from 'toolzy';

console.log(truncate('Hello, world!', 10)); // "Hello, ..."
console.log(truncate('1234567890', 5, '___')); // "12___"

Array Operations

shuffleArray(array) → T[]
▼
Shuffles the elements of the given array in a random order and returns a new array.

Parameters:

  • array (T[]) - The array to be shuffled

Returns:

A new array containing the elements of the input array in random order

Example:
import { shuffleArray } from 'toolzy';

const numbers = [1, 2, 3, 4, 5];
const result = shuffleArray(numbers);
console.log(result); // e.g., [3, 1, 5, 2, 4]
removeDuplicates(array) → T[]
▼
Removes duplicate values from an array, preserving the order of first occurrences.

Parameters:

  • array (array) - The array to deduplicate

Returns:

A new array with duplicates removed.

Example:
import { removeDuplicates } from 'toolzy';

const ids = [1, 2, 2, 3, 1];
console.log(removeDuplicates(ids)); // [1, 2, 3]
intersect(arrayA, arrayB) → array
▼
Returns a new array containing only the elements that are present in both input arrays.

Parameters:

  • arrayA (array) - The first array
  • arrayB (array) - The second array

Returns:

An array of common elements found in both arrays.

Example:
import { intersect } from 'toolzy';

const a = [1, 2, 3];
const b = [2, 3, 4];
console.log(intersect(a, b)); // [2, 3]
difference(arrayA, arrayB) → array
▼
Returns a new array containing elements from the first array that are not present in the second.

Parameters:

  • arrayA (array) - The source array
  • arrayB (array) - The array to exclude from arrayA

Returns:

An array of elements in arrayA that are not in arrayB.

Example:
import { difference } from 'toolzy';

const a = [1, 2, 3];
const b = [2, 3, 4];
console.log(difference(a, b)); // [1]

Date Utilities

formatDate(date, format) → string
▼
Formats a JavaScript Date object into a string based on a custom pattern.

Supported Tokens:

  • Year: YYYY, YY
  • Month: MMMM, MMM, MM, M
  • Day: DD, D, dddd, ddd
  • Hours: HH, H, hh, h
  • Minutes: mm, m
  • Seconds: ss, s
  • Meridiem: A, a

Parameters:

  • date (Date) - The date to format.
  • format (string) - A string pattern to format the date.

Returns:

A string representing the formatted date based on the given pattern.

Example:
import { formatDate } from 'toolzy';

const now = new Date();
console.log(formatDate(now, 'YYYY-MM-DD')); // "2025-05-27"
console.log(formatDate(now, 'ddd, MMM D, YY hh:mm A')); // "Tue, May 27, 25 03:16 PM"
